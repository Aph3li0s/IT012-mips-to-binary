# Đồ án IT012 MIPS to binary code
Build trình thông dịch hợp ngữ MIPS bằng C++ 

* Bao gồm lệnh nhóm lệnh I và R (Không yêu cầu code lệnh J)
* Input file .asm, output file .txt chứa binary code
